# Hitesh Engineering Website

This repository contains the source code for my personal website.

The platform serves as:
- A professional portfolio
- A centralized archive of engineering notes
- A showcase of projects and technical work
- A resource hub for juniors and peers

---

## Website Structure

The website includes:

- **Home** – Introduction and overview
- **About** – Academic background and interests
- **Projects** – Selected technical work
- **Resume** – Downloadable CV
- **Notes** – Structured technical notes categorized by subject

---

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- MDX (for notes and documentation)
- Vercel (deployment)

---

## Purpose

This platform is built to:

- Maintain an independent professional presence
- Organize and document my learning journey
- Share structured notes with juniors
- Present projects in a clean and accessible format
